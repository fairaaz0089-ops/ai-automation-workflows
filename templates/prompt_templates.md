# Prompt Templates

A collection of reusable AI prompt templates for various automation tasks.

---

## Blog Post Generation

```
Write a {word_count}-word SEO-optimized blog post about: {topic}

Requirements:
- Engaging H1 title with target keyword
- Introduction that hooks the reader in 2-3 sentences
- 5-7 H2 subheadings
- Include actionable tips and real examples
- Conclude with a clear call-to-action
- Target keyword: {keyword}
- Tone: {tone} (professional/casual/educational)

Format as markdown.
```

---

## LinkedIn Post

```
Write a LinkedIn post about: {topic}

Requirements:
- Start with a bold hook (no hashtags in first line)
- 3-5 short paragraphs
- Include 1 personal insight or story
- End with a question to drive engagement
- Add 3-5 relevant hashtags at the end
- Tone: thought leadership
- Max 1300 characters
```

---

## Medium Article Intro

```
Write a compelling Medium article introduction about: {topic}

Requirements:
- Start with a surprising statistic or bold statement
- 150-200 words
- Clearly state what the reader will learn
- Create curiosity without giving everything away
- No fluff, get to the point
```

---

## Pinterest Pin Description

```
Write 5 Pinterest pin descriptions for: {topic}

Each description should:
- Be 100-200 characters
- Include a call-to-action
- Have 2-3 relevant keywords naturally embedded
- Be engaging and benefit-focused
- Include relevant emojis
```

## Upwork Proposal

```
Write an Upwork proposal for this job:
{job_description}

My profile:
- Skills: {skills}
- Experience: {years} years
- Portfolio: {portfolio_url}

Requirements:
- Under 150 words
- Lead with understanding their problem
- Mention 1 specific relevant result
- Clear CTA at the end
- No generic openers like "I am interested in..."
```

---

## Email Newsletter

```
Write an email newsletter about: {topic}

Requirements:
- Subject line: under 50 chars, create curiosity
- Preview text: 85-100 chars
- Body: 200-300 words
- 1 main tip or insight
- 1 featured resource or link
- Friendly, conversational tone
- Clear CTA button text
```
