# Workflows

This folder contains Make.com JSON blueprint files ready to import.

## Available Workflows

| File | Description |
|---|---|
| `content-pipeline.json` | Auto-generates and publishes blog posts to WordPress |
| `social-scheduler.json` | Schedules posts to LinkedIn, Medium, and Pinterest |
| `affiliate-tracker.json` | Tracks affiliate link clicks and conversions |
| `upwork-proposals.json` | Monitors Upwork listings and sends auto-proposals |
| `kdp-pipeline.json` | Amazon KDP book creation and upload pipeline |

## How to Import

1. Log into [Make.com](https://make.com)
2. Go to **Scenarios** > **Create a new scenario**
3. Click the **three dots menu** > **Import Blueprint**
4. Upload the desired `.json` file
5. Configure your API credentials in each module
